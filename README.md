# Luciverse API

**A RESTful backend API for Lucifer quotes** built with **Express, Knex, and PostgreSQL**.  
This API provides endpoints to fetch and add quotes and is used by the **Luciverse frontend UI** ([link to frontend here](<https://github.com/hjawad22/luciverse>)).

---

## Features

- `GET /api/quotes` → Retrieve all quotes  
- `POST /api/quotes` → Add a new quote  
- Database setup via **Knex migrations & seeds**  link
